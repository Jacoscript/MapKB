Types of Notifications
  • Error
  • Other

Visual of types
Use Cases
  • You can receive feedback with custom queries for whether they:
    • Worked/Failed
    • Returned any information
  • System wide errors will display so you know when something isn't working
  • Logs for developers
  • Warnings for users/developers
How Notifications are Used When a notification is displayed, we store said notification in a PostgreSQL database for later analysis on the types of problems users may be having when interacting with the UI, the custom query builder, or other components of our system.
What is Logged?
  • Notification message content
  • Time stamp from when the notification occurred
  • Type of notification
  • User agent - Type of browser and version
  • Web address location of when notification occurred
