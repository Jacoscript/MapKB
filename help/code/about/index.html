Frequently Asked Questions (FAQ): About the Code/Project
The Project CEGIS Linked Open Data (LOD) Semantic Technology System is a system for exploring topographic data of The National Map as Resource Description Framework (RDF), a World-Wide Web Consortium (W3C) standard. The central concept of the CEGIS project is: 'The National Map as Knowledge Base'.
Code Overview The code base consists of two branches at the moment, the development branch (up-to-date branch) and the master branch (last major working branch).
Components
  • Apache Tomcat - A tool that manages the user interface and allows others to connect to our server. (Version 8.5.8)
  • Apache Marmotta - A database tool with RDF querying capabilities via SPARQL. The .war version of Tomcat needs to be used. (https://marmotta.apache.org/download.html) (Version 3.4.0 SNAPSHOT)
  • Apache Maven - A tool that runs the Karma Web App. (https://maven.apache.org/download.cgi) (Version 3.6.0)
  • GeoServer - A server that allows us to manage WFS endpoints and data. The .war version of this needs to be used for Tomcat. (http://geoserver.org/download/) (Version 2.16.0)
  • Karma-As-A-Service - A tool used in coordination with Marmotta to act as an endpoint for its queries. This may all come together as a single download with the Karma Web App. (https://github.com/usc-isi-i2/Web-Karma) (Version 0.0.1 SNAPSHOT)
  • Karma Web App - A tool used to perform data manipulation when pulling data from the GeoServer. (https://github.com/usc-isi-i2/Web-Karma) (Version 2.051)
  • MAKB Web Tool - A user interface for the project that needs to be deployed on Tomcat. This user interface is within the current repository. (https://code.chs.usgs.gov/cegis/makb/tree/master) (Version 1.0)
  • PostgreSQL - A tool used for the LD (linked data) project. (https://www.postgresql.org/download/) (Version 9.6)
Not Actively Used Components
  • GDAL - A tool used to convert data between multiple formats in bulk. Such formats are .txt to .shapefile conversion. (https://gdal.org/download.html) (Version 3.0.0)
  • LIMES - A tool used to create sameAs relations. (https://github.com/dice-group/LIMES) (Version 1.4.0 and 1.5.0?)
